Privacy Policy - TROI Improved

Last updated: 2024

Overview

TROI Improved is a Chrome extension that enhances the user interface of TROI Software by adding German text labels to buttons and optimizing layout spacing.

Data Collection and Usage

This extension:

  • Does not collect any user data
  • Does not transmit any information
  • Does not track user behavior
  • Does not store any personal information

Host Permissions

The extension requires access to *.troi.software/* domains solely to:

  • Add German text labels to buttons through CSS
  • Optimize spacing between UI elements
  • Apply visual improvements to the interface

No data is collected, stored, or transmitted during these operations.

Datenschutzerklärung - TROI Improved

Zuletzt aktualisiert: 2024

Überblick

TROI Improved ist eine Chrome-Erweiterung, die die Benutzeroberfläche von TROI Software durch das Hinzufügen deutscher Textbeschriftungen zu Schaltflächen und die Optimierung von Layoutabständen verbessert.

Datenerfassung und -nutzung

Diese Erweiterung:

  • Sammelt keine Benutzerdaten
  • Überträgt keine Informationen
  • Verfolgt kein Benutzerverhalten
  • Speichert keine persönlichen Informationen

Host-Berechtigungen

Die Erweiterung benötigt Zugriff auf *.troi.software/* Domains ausschließlich um:

  • Deutsche Textbeschriftungen mittels CSS zu Schaltflächen hinzuzufügen
  • Abstände zwischen UI-Elementen zu optimieren
  • Visuelle Verbesserungen der Oberfläche vorzunehmen

Bei diesen Vorgängen werden keine Daten gesammelt, gespeichert oder übertragen.
